# Generating beaver triples using homomorphic encryption

Решение 3 дз, реализация протокола генерации троек Бивера с использованием гомоморфного шифрования и бибилиотеки *phe*.
Коммуникация между сторонами организована через **`torch.distributed`**.

## Архитектура

Система состоит из двух Docker-контейнеров:


- **Party0 и Party1** — участники протокола:  
  - считывают свои доли чисел и локальные Beaver-тройки;  

---

## Запуск

Количество генерируемых троек Бивера задается в docker-compose.yml
Результат сохраняется в ./triples (путь в mul.py уже изменен на корректный для данной реализации)

Построить и запустить контейнеры:

```bash
cd homework
docker-compose up --build
```

Выполнить проверку корректности сгенерированных троек Бивера:

```bash
cd homework
python mul.py
