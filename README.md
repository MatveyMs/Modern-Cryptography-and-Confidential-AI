## The repository contains completed homework assignments from the T-bank course: "Modern Cryptography and Confidential AI"

### 1 task
Learning linear regression in CrypTen by MPC. Performs two scenarios for learning a linear model:
1) linreg, when one side contains features and the other contains a target parameter
2) linreg, when both parties contain features and a target parameter and want to train the model together

### 2 task
Implementation of a protocol for comparing two numbers using Beaver triples

### 3 task
Implementation of the Beaver triple generation protocol using homomorphic encryption and the phe library
